# 🌦️ Weather Type Prediction using Machine Learning

## 📌 Project Overview

This project is a **Machine Learning-based Weather Type Prediction System** that predicts the type of weather based on environmental conditions such as temperature, humidity, wind speed, season, and location.

Unlike traditional weather apps that rely on APIs, this project uses a trained ML model to **learn patterns from historical data and make predictions**.

---

## 🎯 Objectives

* Predict weather type using environmental features
* Build an end-to-end ML pipeline
* Deploy the model as an interactive web application

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* Pickle

---

## 📂 Dataset Features

* Temperature
* Humidity
* Wind Speed
* Season
* Location
* Other environmental parameters

---

## ⚙️ Project Workflow

### 1. Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature scaling using StandardScaler

### 2. Handling Class Imbalance

* Applied techniques to balance dataset

### 3. Model Building

* Used classification algorithms from Scikit-learn

### 4. Model Evaluation

* Evaluated model using accuracy and other metrics

### 5. Model Saving

* Saved model, scaler, and encoders using Pickle

### 6. Deployment

* Built an interactive web app using Streamlit
